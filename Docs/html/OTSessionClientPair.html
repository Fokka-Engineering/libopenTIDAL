NAME

OTSessionClientPair - Specify the OAuth2 clientId and clientSecret

SYNOPSIS

#include <openTIDAL/openTIDAL.h>

int OTSessionClientPair (const char *const clientId, const char *const clientSecret);

DESCRIPTION

libopenTIDAL(3) uses TIDALs OAuth2 authorisation service to authenticate with the TIDAL API. For this to work you need a clientId and a clientSecret.

Note: The clientSecret is only required if one plans to use the auth and user-based services.

The device authorisation grant is not enabled in all clientIds. You need a clientId/clientSecret pair that was generated to be used in restricted-input or browserless devices/applications. Standard webflow pairs will throw an error.

Features like preview track/video streams are only available in the restrictedMode with the web clientId/clientSecret pair.

RETURN VALUE

If successful OTSessionClientPair returns 0.

If an MALLOC error occurred OTSessionClientPair returns -1.
