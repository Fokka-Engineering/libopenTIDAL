NAME

OTJsonGetObjectItemNumberValue - Get an object and its number

SYNOPSIS

#include <openTIDAL/openTIDAL.h>

double OTJsonGetObjectItemNumberValue (const struct OTJsonContainer *const object, const char *const string);

DESCRIPTION

Get a number from an object that represents a json number. Use the parent object and the string parameter to find the object.

RETURN VALUE

If the object is a valid number, a double gets returned. Otherwise, a NaN gets returned.
