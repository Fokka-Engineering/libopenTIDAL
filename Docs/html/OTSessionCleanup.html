NAME

OTSessionCleanup - End a libopenTIDAL session handle

SYNOPSIS

#include <openTIDAL/openTIDAL.h>

void OTSessionCleanup (struct OTSessionContainer *session);

DESCRIPTION

This function must be the last function to call for a libopenTIDAL(3) session. It deallocates the memory allocation performed by the OTSessionInit function. It is the opposite of the init function and must be called with the same session as input that a OTSessionInit call allocated.

Do not call this function if you intend to call more libopenTIDAL(3) service functions.

Any use of the session or the libopenTIDAL(3) service functions without reinitialising is illegal. OTSessionCleanup(3) deallocates all memory associated with the session and kills the curl handle.

RETURN VALUE

None

EXAMPLE

struct OTSessionContainer *session = OTSessionInit ();
if (session)
    {
        OTSessionClientPair ({clientId}, {clientSecret}); 
        /* do smth with the session if OTSessionClientPair returns 0 */
    }
OTSessionCleanup (session);
