NAME

OTQuality - Streaming quality enums

DESCRIPTION

This man page includes all available streaming quality codes in libopenTIDAL. Their corresponding codecs and bitrates/samplerates are also included.

OTQuality

The actual bitrate/framerate or resolution might differ depending on the used clientId or the requested artefact.

AUDIO_LOW (0)

Advanced Audio Coding (AAC+) @ 96 kbps.

AUDIO_HIGH (1)

Advanced Audio Coding (AAC) @ 320 kbps.

AUDIO_LOSSLESS (2)

Free Lossless Audio Codec (FLAC) @ 1411 kbps, 16bit/44.1kHz.

AUDIO_HI_RES (3)

Free Lossless Audio Codec (FLAC) + MQA Encoding @ 24bit/96kHz.

VIDEO_AUDIO_ONLY (4)

High-Efficiency Advanced Audio Coding (AAC, mp4a.40.5).

VIDEO_LOW (5)

Low-Complexity Advanced Audio Coding (AAC, mp4a.40.2) with H264 video (avc1.42001e) @ 320x180 25fps.

VIDEO_MEDIUM (6)

Low-Complexity Advanced Audio Coding (AAC, mp4a.40.2) with H264 video (avc1.4d001f) @ 640x360 25fps.

VIDEO_HIGH (7)

HTTP Live Streaming (HLS) H264 with multiple video and audio feeds. From 1920x1080 25fps @ 10173 kbps to 320x180 12.5fps @ 318 kbps.
