NAME

OTJsonGetStringValue - Get string from object

SYNOPSIS

#include <openTIDAL/openTIDAL.h>

char *OTJsonGetStringValue (const struct OTJsonContainer *const item);

DESCRIPTION

Get an ASCII string from an object that represents a json string. Instead of calling OTJsonGetObjectItem use OTJsonGetObjectItemStringValue to find the object. DO NOT deallocate the ASCII string manually. OTDeallocContainer handles OTJsonContainer tree deallocation by iterating over the tree and its values.

RETURN VALUE

If the object is a valid string an ASCII string gets returned. Otherwise, a NULL pointer gets returned.
