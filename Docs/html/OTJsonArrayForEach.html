NAME

OTJsonArrayForEach - Iterate over an array

SYNOPSIS

#include <openTIDAL/openTIDAL.h>

#define OTJsonArrayForEach(element, array) for (element = (array != NULL) ? (array)->child : NULL; element != NULL; element = element->next)

DESCRIPTION

An array is stored as a linked list, iterating it via OTJsonGetArraySize and OTJsonGetArrayItem is inefficient (O(n2)), so you can iterate over an array using the OTJsonArrayForEach macro in O(n) time complexity.

RETURN VALUE

None
