NAME

OTServiceMovePlaylistItem - Move an artefact in a playlist

SYNOPSIS

#include <openTIDAL/openTIDAL.h>

enum OTStatus OTServiceMovePlaylistItem (struct OTSessionContainer *session, const char *const id, const int index, const int toIndex, void *threadHandle);

DESCRIPTION

The OTServiceMovePlaylistItem service function moves an artefact in a playlist.

A playlist starts from 0.

RETURN VALUE

OTStatus
