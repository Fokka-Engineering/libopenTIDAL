NAME

OTJsonIsObject - Check for an object value

SYNOPSIS

#include <openTIDAL/openTIDAL.h>

int OTJsonIsObject (const struct OTJsonContainer *const item);

DESCRIPTION

Represents an object value. Check for an object value.

Objects are stored same way as an array, the only difference is that the items in the object store their keys in string OTJsonContainer.

RETURN VALUE

Returns either 1 or 0 as an integer value.
