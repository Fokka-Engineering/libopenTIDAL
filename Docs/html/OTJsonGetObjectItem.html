NAME

OTJsonGetObjectItem - Access an item in an object

SYNOPSIS

#include <openTIDAL/openTIDAL.h>

struct OTJsonContainer *OTJsonGetObjectItem (const struct OTJsonContainer *const object, const char *const string);

DESCRIPTION

If you want to access an item in an object, use this function. Use the parent object and the string parameter to find the object. This function is standard compliant and follows the JSON standard. It's case sensitive.

RETURN VALUE

OTJsonContainer
