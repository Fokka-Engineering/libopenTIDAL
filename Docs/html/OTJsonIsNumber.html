NAME

OTJsonIsNumber - Check for a number value

SYNOPSIS

#include <openTIDAL/openTIDAL.h>

int OTJsonIsNumber (const struct OTJsonContainer *const item);

DESCRIPTION

Represents a number value. Check an object for a json number value.

The value is stored as a double in valuedouble and also in valueint OTJsonContainer. If the number is outside of the range of an integer, INT_MAX or INT_MIN are used for valueint.

RETURN VALUE

Returns either 1 or 0 as an integer value.
