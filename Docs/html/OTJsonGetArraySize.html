NAME

OTJsonGetArraySize - Get the size of an array

SYNOPSIS

#include <openTIDAL/openTIDAL.h>

int OTJsonGetArraySize (const struct OTJsonContainer *array);

DESCRIPTION

Call this function to get the size of an array or object, this works because internally objects are stored as arrays. Check if the OTJsonContainer represents an array with OTJsonIsArray.

RETURN VALUE

Returns the number of items in an array as an integer value.
