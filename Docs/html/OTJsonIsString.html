NAME

OTJsonIsString - Check for a string value

SYNOPSIS

#include <openTIDAL/openTIDAL.h>

int OTJsonIsString (const struct OTJsonContainer *const item);

DESCRIPTION

Represents a string value. Check an object for a json string value.

It is stored in the form of a zero terminated string in valuestring OTJsonContainer.

RETURN VALUE

Returns either 1 or 0 as an integer value.
