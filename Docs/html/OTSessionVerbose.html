NAME

OTSessionVerbose - Enable the verbose mode

SYNOPSIS

#include <openTIDAL/openTIDAL.h>

void OTSessionVerbose (struct OTSessionContainer *const session, const int enabled);

DESCRIPTION

Toggle openTIDAL(3)'s verbose feature. This enables libcurl CURLOPT_VERBOSE and the internal verbose printf calls. openTIDAL(3) prints to stdout.

RETURN VALUE

None
