NAME

OTJsonGetNumberValue - Get a number from object

SYNOPSIS

#include <openTIDAL/openTIDAL.h>

double OTJsonGetNumberValue (const struct OTJsonContainer *const item);

DESCRIPTION

Get a number from an object that represents a json number. Instead of calling OTJsonGetObjectItem use OTJsonGetObjectItemNumberValue to find the object.

RETURN VALUE

If the object is a valid number, a double gets returned. Otherwise, a NaN gets returned.
