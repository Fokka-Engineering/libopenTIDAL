NAME

OTServiceDeleteFavorite - Manipulate favorites

SYNOPSIS

#include <openTIDAL/openTIDAL.h>

enum OTStatus OTServiceDeleteFavorite (struct OTSessionContainer *session, const char *const suffix, const char *const id, void *threadHandle);

DESCRIPTION

The OTServiceDeleteFavorite service function deletes an artefact from the user-saved content.

To delete an artefact from the collection, call the correct endpoint that represents the type of the artefact. 

Suffixes

"albums", "tracks", "videos", "artists", "playlists", "mixes" 

Thread Handle

You must never share the same handle in multiple threads. You can pass the handles around among threads, but you must never use a single handle from more than one thread at any given time.

Use the session main handle by parsing a NULL pointer.

RETURN VALUE

OTStatus
