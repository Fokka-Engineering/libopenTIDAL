NAME

OTJsonGetObjectItemStringValue - Get an object and its string

SYNOPSIS

#include <openTIDAL/openTIDAL.h>

char *OTJsonGetObjectItemStringValue (const struct OTJsonContainer *const object, const char *const string);

DESCRIPTION

Get an ASCII string from an object that represents a json string. Use the parent object and the string parameter to find the object. DO NOT deallocate the ASCII string manually. OTDeallocContainer handles OTJsonContainer tree deallocation by iterating over the tree and its values.

RETURN VALUE

If the object is a valid string, an ASCII string gets returned. Otherwise, a NULL pointer gets returned.
