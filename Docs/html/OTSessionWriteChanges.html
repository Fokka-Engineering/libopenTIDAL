NAME

OTSessionWriteChanges - Write session changes to the persistent config

SYNOPSIS

#include <openTIDAL/openTIDAL.h>

int OTSessionWriteChanges (const struct OTSessionContainer *session);

DESCRIPTION

If one makes changes to the libopenTIDAL(3) session handle and its allocated structure itself, calling the OTSessionWriteChanges function will make these changes persistent.

libopenTIDAL(3) overwrites the old persistent config. The file location path was specified prior in the OTSessionLogin function parameter.

RETURN VALUE

Status (0) means everything was ok. If a parsing error occurred Status (-1).

SEE ALSO

OTSessionChangeQuality(3), OTSessionCleanup(3), OTSessionRefresh(3)
