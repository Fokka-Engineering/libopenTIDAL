NAME

OTHttpThreadHandleCleanup - Free a http-handle

SYNOPSIS

#include <openTIDAL/openTIDAL.h>

void OTHttpThreadHandleCleanup (void *handle);

DESCRIPTION

This function must be the last function to call in a thread.

It is the opposite of the init function and must be called with the same session as input that a OTHttpThreadHandleCreate call allocated.

RETURN VALUE

None
