NAME

OTJsonGetArrayItem - Get an element at a given index

SYNOPSIS

#include <openTIDAL/openTIDAL.h>

struct OTJsonContainer *OTJsonGetArrayItem (const struct OTJsonContainer *array, int index);

DESCRIPTION

Call this function to get an element at a given index of an array. Check if the OTJsonContainer represents an array with OTJsonIsArray.

Because an array is stored as a linked list, iterating it via index is inefficient (O(n2)), so you can iterate over an array using the cJSON_ArrayForEach(3) macro in O(n) time complexity.

RETURN VALUE

OTJsonContainer
