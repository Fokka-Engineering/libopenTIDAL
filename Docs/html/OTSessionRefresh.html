NAME

OTSessionRefresh - Request a manual session renewal

SYNOPSIS

#include <openTIDAL/openTIDAL.h>

enum OTStatus OTSessionRefresh (struct OTSessionContainer *session);

DESCRIPTION

Call this function to manually trigger an session renewal. This can be useful if a service function returns EXPIRED_SESSION in the main thread.

This only occurs if the internal call fails. The internal refresh call is not likely to fail.

RETURN VALUE

Returns an enum OTStatus.
