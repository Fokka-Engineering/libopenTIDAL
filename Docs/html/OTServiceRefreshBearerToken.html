NAME

OTServiceRefreshBearerToken - Request a new access-token from the refresh-token

SYNOPSIS

#include <openTIDAL/openTIDAL.h>

struct OTContentContainer *OTServiceRefreshBearerToken (struct OTSessionContainer *session, const char *const refreshToken, void *threadHandle);

DESCRIPTION

Exchange a refresh token for an access token when the access token has expired.

The server does not issue a new refresh token in the response. This service function should not be called by the application since it is integrated in the http-connector.

This call must have a corresponding call to OTDeallocContainer when the operation is complete. 

Thread Handle

You must never share the same handle in multiple threads. You can pass the handles around among threads, but you must never use a single handle from more than one thread at any given time.

Use the session main handle by parsing a NULL pointer.

RETURN VALUE

If no memory allocation error occurred in allocating the OTContentContainer, a pointer to an OTContentContainer will be returned. Otherwise a NULL pointer will be returned.
